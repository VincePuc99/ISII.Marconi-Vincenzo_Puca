07-Corso sugli script bash - aritmetica semplice

Negli script bash si possono effettuare delle semplice operazioni aritmetiche.
Per esempio la moltiplicazione: echo $((2*3))

L'artitmetica negli script bash è solo per numeri interi, quindi se si effettua ad esempio una divisione non si vedranno i decimali.
Però c'è il comando "bc" che può farlo. Richiamando il comando bc con l'opzione " l " otterremo anche i decimali.
Un paio di esempi (l'opzione "scale" indica quanti decimali vogliamo):

  • echo "scale=5; 2/3" | bc -l ⇒ 0.66666
  • A=2, B=3 → var1=$(echo "scale=10; $A/$B" | bc -l) ⇒ 0.6666666666

Anche con le tuple si possono fare operazioni aritmetiche:
Numeri[5]=`expr ${Numeri[1]} + ${Numeri[3]}`
sintassi equivalente a
Numeri[5]=$((${Numeri[1]} * ${Numeri[3]}))

nota1: il comando bc può fare molto di più, anche funzioni trigonometriche e logaritmi tanto per fare degli esempi. È come avere a disposizione una calcolatrice scientifica.
nota2: il comando expr è definito come un valutatore di espressioni per tutti gli usi molto versatile (aritmetica, stringhe, logica, confronti)
