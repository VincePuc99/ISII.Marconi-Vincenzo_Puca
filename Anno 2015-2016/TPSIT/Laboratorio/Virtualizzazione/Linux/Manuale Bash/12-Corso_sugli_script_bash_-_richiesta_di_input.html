12-Corso sugli script bash - richiesta di input

Il comando read permette di interrompere l'esecuzione di uno script in attesa dell'inserimento da parte dell'utente di un input, ovvero di un'informazione necessaria al prosieguo dell'elaborazione. Con questo comando possiamo aggiungere l'interattività al nostro script che finora era un programma eseguito dall'inizio alla fine senza intervento alcuno, a meno di modificare il codice prima di rieseguirlo.
Il comando read non richiede per forza che si imposti qualcosa, ma può anche essere usato semplicemente per introdurre una pausa nell'esecuzione dello script. Basta un enter per proseguire. Questo comando l'abbiamo già incontrato nelle lezioni precedenti usato proprio a questo scopo.
In altre parole, se finora lo script doveva avere tutte le impostazioni corrette prima di essere eseguito, perché una volta partito non potevamo più influire sull'esito in alcun modo, ora possiamo far partire l'esecuzione che ci chiederà cammin facendo le informazioni che gli servono.
Inoltre ad ogni esecuzione dello script possiamo fornirgli informazioni diverse, determinando di volta in volta un esito ed un comportamento diverso. E questo è davvero lo scopo finale della programmazione. Avere uno strumento a nostra disposizione sufficientemente dinamico ed adattabile da soddisfare tutte le nostre esigenze.

  • read + ENTER (pausa nello script)
  • read v1 v2 ... vn (da uno a più valori con un solo comando)
  • read -a tupla (inserisce i valori forniti dall'utente in una tupla)
  • read -r rigatesto (legge un file di testo, riga per riga, ed esegue le operazioni necessarie)
  • read -s -n5 input (-s → non visualizza i caratteri digitati, stile password -n5 → accetta 5 caratteri e poi prosegue l'elaborazione)

Intercettazione di tasti speciali (F1 .. F12, ENTER, frecce tastiera)

  • Il tasto ENTER si può testare semplicemente confrontando se la stringa ritornata è vuota
  • altri tasti speciali come F12 sono intepretati come sequenze composte dal carattere speciale escape e da una serie di caratteri. Per determinare la sequenza premere CTRL+V e poi il tasto speciale
    • per esempio il tasto F12 corrisponde alla sequenza: ^[[24~ dove "^[" è il carattere escape
  • il carattere escape ha questo codice esadecimale che si può usare negli script Escape=$'\x1b'. Quindi il tasto F12 completo sarebbe: F12=$Escape"[24~"
